Overview
  This is a Java-based ATM console application.
It allows users to login, create accounts, and perform basic banking operations like checking balance, deposit, withdraw, and transfer money.

Features
  Login using Customer Number & PIN
  Create a new bank account
  Access:
  Checking Account
  Savings Account
  View account balance
  Deposit money
  Withdraw money
  Transfer funds between accounts
  Input validation for wrong entries

Technologies Used
  Java
  HashMap
  Scanner (for user input)
  Exception Handling
  OOP Concepts
